# ema-rsi-backtester
EMA + RSI trading strategy backtester with Order Blocks visualization (Python, pandas, Plotly)

# Індивідуальний проєкт: Бектестер стратегії та індикатор Order Blocks

У цьому проєкті я реалізував бектестер для перевірки прибутковості торгової стратегії на історичних даних, а також переписав індикатор Order Blocks з PineScript на Python для візуалізації ринкових структур.

## Огляд системи

Весь код написаний в ООП стилі для зручності підтримки та розширення. Основна логіка розділена на два класи:

### 1. `IndicatorEngine`
Цей клас відповідає за технічні розрахунки. Я реалізував індикатори на чистому `pandas`, щоб не залежати від сторонніх бібліотек на кшталт `pandas_ta`.
- **calculate_ema**: Рахує експоненційну середню для визначення тренду.
- **calculate_rsi**: Рахує індекс відносної сили для пошуку точок розвороту.

### 2. `IndividualProject`
Це ядро програми, яке об'єднує всі процеси.
- **get_historical_data**: Тягне хвилинні свічки через Yahoo Finance API.
- **find_order_blocks**: Шукає на графіку зони Supply (пропозиції) та Demand (попиту).
- **run_backtest**: Основний цикл, який проходить по кожній свічці та імітує реальну торгівлю з перевіркою умов входу та виходу (TP/SL).
- **visualize**: Малює інтерактивний графік зі свічками, індикаторами та знайденими блоками.

## Моя стратегія (TFMS)
Для проєкту я вибрав стратегію **Trend-Follow Momentum Scalping**. Суть проста: торгуємо по тренду, але заходимо тільки на відкатах.
- **Long**: Коли EMA(20) вище за EMA(50) (тренд вгору) і RSI падає нижче 40 (ціна перепродана на відкаті).
- **Short**: Коли EMA(20) нижче за EMA(50) (тренд вниз) і RSI піднімається вище 60 (ціна перекуплена на відкаті).
- **Ризик-менеджмент**: Використовую Take Profit 1.5% та Stop Loss 0.7%.
